# Maxi80

The Maxi80 radio streaming app — built with Swift and SwiftUI, targeting iOS, iPadOS, and macOS.

This repository replaces:
- [`sebsto/maxi80-ios-swift`](https://github.com/sebsto/maxi80-ios-swift) (archived)
- [`sebsto/maxi-80-android-exoplayer2`](https://github.com/sebsto/maxi-80-android-exoplayer2) (archived)

## About

Maxi80 is a live radio streaming app for [maxi80.com](https://maxi80.com), playing the best hits from the 80s. The app provides live audio streaming with real-time metadata display (artist, song title, album art).

## Tech Stack

- Swift 6 / SwiftUI
- AVFoundation for audio streaming
- Async/await concurrency
- Targets: iOS 18+, iPadOS 18+, macOS 15+

## Getting Started

Open `Maxi80.xcodeproj` in Xcode 16+ and build.
